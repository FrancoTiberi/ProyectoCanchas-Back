# ProyectoCanchas - Backend REST API ⚽

API RESTful desarrollada para gestionar el backend de un sistema de reservas de la página Golazo Gourmet. Se encarga de la lógica de negocio, la gestión de usuarios, el control de disponibilidad de horarios y el procesamiento de pagos online.

## Tecnologías Utilizadas

* **Entorno:** Node.js
* **Framework:** Express.js
* **Base de Datos:** MongoDB
* **Pasarela de Pagos:** Mercado Pago (Checkout Pro y Webhooks)
* **Despliegue:** Vercel

## Ejecución y Instalación del proyecto

1. **Clonar el repositorio:**
   Escribir en la terminal el siguiente comando:
   ```bash
   git clone https://github.com/FrancoTiberi/ProyectoCanchas-Back
2. **Instalar dependencias:**
